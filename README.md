> Use node version `16.0.0` or above

## Scripts

---------

#### Run the app
##### `npm start [filepath]`
##### *e.g. `npm run input.json`*

----

#### Run the app in watch mode
##### `npm run dev [filepath]`
##### *e.g. `npm run dev input.json`*

----

#### Run tests
##### `npm test`

##### *you can use `test:cover` and `test:watch` also*

----

#### Run linting
##### `npm run lint`

----

#### Auto fix linting issues
##### `npm run lint:fix`
